Q1: Devs and Testers, encompassing unit, component, and API testing.

Q2: Tests that are business facing and primarily executed by testers, e.g. functional, UI, and regression testing.

Q3: Tests business facing but rely on external dependencies such as database, web services, integrations.

Q4: Tests carried out by stakeholders, BA, or users, e.g. usability testing, acceptance testing, and exploratory testing. Evaluates software's user experience and how it aligns with business needs and user experience.
